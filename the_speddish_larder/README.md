## App Overview

**The Speddish Larder** is a comprehensive kitchen and cooking assistant mobile application that includes:

- **Recipe Management**: Create, store, and organize personal recipes
- **Shopping Lists**: Collaborative shopping lists with real-time sharing
- **Pantry Tracking**: Monitor inventory, expiration dates, and get smart suggestions
- **Meal Planning**: Plan meals with calendar integration and automated shopping list generation
- **AI Assistance**: Get personalized recipe suggestions and cooking tips

## Technical Details

- **Platform:** Android and iOS (Flutter)
- **Authentication:** Google Sign-In
- **Backend:** Firebase (Firestore for data, Authentication for users)
